# Workflow: X-RAY-ALTAS - Decision Intelligence

## Objetivo
Fornecer um sistema de Decision Intelligence para fundadores sob incerteza, utilizando 5 modos cognitivos e uma Base de Conhecimento (KB) estruturada. Apoiar a tomada de decisões estratégicas em ambientes complexos.

## ICP (Ideal Customer Profile)
Solo founders, tech CEOs, strategic operators.

## Fases do Workflow

### Fase 1: Enquadramento do Desafio (What)
*   **O Quê:** Definição clara do desafio ou decisão a ser tomada.
*   **Como:** Interação com o usuário para contextualizar o problema e identificar as variáveis chave.
*   **Output:** Declaração do problema e critérios de sucesso para a decisão.

### Fase 2: Ativação dos Modos Cognitivos (How)
*   **O Quê:** Aplicação de 5 modos cognitivos para analisar o desafio sob diferentes perspectivas.
*   **Como:** Utiliza skills especializadas para cada modo (ex: análise de cenários, pensamento lateral, análise de riscos, etc.).
*   **Output:** Insights multifacetados e identificação de pontos cegos.

### Fase 3: Consulta à Base de Conhecimento (Where)
*   **O Quê:** Busca e recuperação de informações relevantes da KB estruturada ALTAS.
*   **Como:** Skills de recuperação de informação e síntese de conhecimento.
*   **Output:** Dados e referências contextuais para embasar a decisão.

### Fase 4: Síntese e Recomendação (Why)
*   **O Quê:** Consolidação dos insights e geração de recomendações acionáveis.
*   **Como:** Skills de síntese e formulação de estratégias.
*   **Output:** Relatório de recomendação de decisão com justificativas e potenciais impactos.

### Fase 5: Monitoramento e Ajuste (When)
*   **O Quê:** Acompanhamento da decisão implementada e ajustes conforme necessário.
*   **Como:** Skills de monitoramento de performance e feedback loop.
*   **Output:** Relatório de performance da decisão e sugestões de otimização.

## Skills Envolvidas (Exemplos)
*   `xray-altas-hunter-v4.1`
*   `xray-altas-corpus-analyst-v1`

## Pontos de Controle
*   Clareza do desafio e critérios de decisão.
*   Abrangência da análise pelos modos cognitivos.
*   Relevância e precisão das informações da KB.
*   Coerência e acionabilidade das recomendações.

## Interdependências
*   Pode consumir outputs de `X-RAY-ANALYTICS` para dados estratégicos.
*   Pode gerar inputs para `X-RAY-EXECUTIVE` para implementação de planos.
