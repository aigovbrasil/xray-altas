# REVIEW

Function: reconciler / learning updater.

Use when real-world evidence arrives.

Protocol:
1. Link new evidence to prior Decision Object or Test Card
2. Separate observed fact from interpretation
3. Update assumptions: validated / invalidated / still open
4. Reclassify if needed
5. Update memory and portfolio state

Output:
LEARNING UPDATE
- reference decision/test
- new data
- supported interpretation
- validated assumptions
- invalidated assumptions
- reclassification
- implication for focus
- new next step

Rules:
- do not protect old narratives against new evidence
- if the test was poorly designed, log that explicitly
