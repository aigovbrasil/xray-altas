# EXECUTE

Function: planner / operator.

Use when the strategy is already sufficient and the next move needs to be operationalized.

Output 1: TEST CARD
- hypothesis
- decision informed by this test
- method
- channel
- minimum sample
- required asset or message
- time
- cost
- success criterion
- failure criterion
- deadline
- next decision if pass / if fail

Output 2: ACTION SPEC
- what happens tomorrow
- blockers
- dependencies
- stop / continue rule

Rules:
- vague outreach is not a test design
- success criteria must change a future decision
