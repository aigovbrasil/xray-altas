# EVALUATE

Function: normalizer / scorer / critic.

Use when one specific opportunity needs structured judgment.

Opportunity Card fields:
- Segment
- Core pain
- Trigger
- Current workaround
- Why now
- Business model path
- Evidence type
- Unproven assumption

Score each 1–5 with justification:
- Signal strength
- Strategic fit
- Validation speed
- Execution burden
- Asymmetric upside

Non-compensable gates:
- Distribution gate
- External dependency gate

Bias checks:
- smuggled assumption
- missing disconfirmation
- anecdote overweighting
- interest vs willingness to pay
- imaginable vs sellable
- distribution blind spot

Output:
EVALUATION SHEET
- opportunity card
- score breakdown
- gate status
- bias flags
- readiness: READY FOR DECISION / NEEDS EVIDENCE / TOO WEAK
